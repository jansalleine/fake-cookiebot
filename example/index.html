FakeCookiebot.js is just a little helper script to, well, fake a Cookiebot.

It's sole purpose is to be used in development when our customers didn't have a Cookiebot license yet, but we knew they would buy one and we already wanted so implement frontend styling and functionality.

It just throws a "CookiebotOnAccept" event, when called with:
Cookiebot.renew();
and provides very limited, basic Cookiebot features.

The main function is wrapped in an
if (typeof(window.Cookiebot) === "undefined").
So in case you forget to remove it in production it will not fire when your real Cookiebot is defined first.

In this example the event is handled like this:

"use strict";

(function (d, w)
{
    var styleElem = d.createElement("style");

    styleElem.innerHTML =
        ".cookieconsent-optout-marketing { display: none !important; }";

    w.addEventListener(
        "CookiebotOnAccept",
        function (event)
        {
            if (Cookiebot.consent.marketing)
            {
                d.head.append(styleElem);
            }

            if (!Cookiebot.consent.marketing)
            {
                styleElem.remove();
            }
        },
        false
    );
}
)(document, window);
